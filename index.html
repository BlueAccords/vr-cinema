Cinemality was a project presented at NBC/Universal Hackathon in Sunnyvale, California.
Made by Ilan Granot and Ryan Tran, you can find the full source of the 3D VR player and the server on Github.
For best experience use with a Google cardboard and a last generation mobile phone with the latest version of Mozilla Firefox Beta (currently only works with Firefox beta for Android which supports the 3D video projection on the mobile phone browser).
